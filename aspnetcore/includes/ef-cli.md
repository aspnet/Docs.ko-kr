`dotnet ef`가 설치되지 않은 경우 전역 도구로 설치합니다.

```dotnetcli
  dotnet tool install --global dotnet-ef
```

EF Core의 CLI에 대한 자세한 내용은 [.Net CLI에 대한 EF Core 도구 참조](/ef/core/miscellaneous/cli/dotnet)를 참조하세요.
