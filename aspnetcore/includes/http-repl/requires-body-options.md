* `-c|--content`

  인라인 HTTP 요청 본문을 제공합니다. 예들 들어 `-c "{"id":2,"name":"Cherry"}"`입니다.

* `-f|--file`

  HTTP 요청 본문을 포함하는 파일에 대한 경로를 제공합니다. 예들 들어 `-f "C:\request.json"`입니다.

* `--no-body`

  HTTP 요청 본문이 필요하지 않음을 나타냅니다.
